# 胶片颗粒 (Film Grain) 特性研究

> 研究日期: 2026-02-20

## 一、基本定义

**胶片颗粒** (Film Grain / Film Granularity) 是处理后的摄影胶片上的随机光学纹理。

- **黑白胶片**：由微小的金属银颗粒形成
- **彩色胶片**：由彩色染料云 (dye clouds) 形成

当卤化银接收到足够的光子后，会产生随机分布的小金属银微粒，由此在图像上造成的一种光学质感。

---

## 二、核心特征

### 2.1 随机性与有机感

| 特性 | 描述 |
|------|------|
| **随机分布** | 颗粒在画面中随机分布，非规律性排列 |
| **帧间变化** | 每帧颗粒位置略有不同，产生有机运动感 |
| **非均匀性** | 与数字噪声的均匀分布不同，更具自然感 |

### 2.2 ISO 感光度与颗粒关系

| ISO 等级 | 晶体大小 | 颗粒表现 | 适用场景 |
|----------|----------|----------|----------|
| 低 ISO (50-200) | 小晶体 | 细腻、平滑色调过渡 | 光线充足环境 |
| 中 ISO (400-800) | 中等晶体 | 适度颗粒感 | 通用场景 |
| 高 ISO (1600+) | 大晶体 | 明显粗颗粒 | 低光环境 |

**规律**：ISO 越高，银盐晶体越大，颗粒越明显。

### 2.3 黑白 vs 彩色胶片颗粒差异

| 特性 | 黑白胶片 | 彩色胶片 |
|------|----------|----------|
| **颗粒形态** | 硬边晶体 (hard edges) | 软边染料云 (soft-edged dye clouds) |
| **视觉锐度** | 看起来更锐利 | 相对柔和 |
| **单色区域** | 纯粹的明暗变化 | 出现多彩色点 |
| **颗粒结构** | 金属银微晶体 | 有机染料长分子 |

### 2.4 颗粒尺寸分布

| 分布类型 | 特点 | 优势 |
|----------|------|------|
| **窄分布 (Monosize)** | 晶体尺寸均匀一致 | 更好的感光度控制，颗粒更细腻 |
| **宽分布 (Random)** | 晶体尺寸随机变化 | 曝光和冲洗容错度更高 |

## 三、影响颗粒可见度的因素

### 3.1 光线条件
- **充足光线** (日光/柔和棚光)：颗粒不明显
- **低光环境**：颗粒更加突出

### 3.2 曝光程度
- **正常/过曝**：颗粒较少
- **欠曝**：暗部颗粒更明显

### 3.3 胶片品牌特征

不同品牌有独特的"颗粒签名"：
- **Kodak**：温暖色调，颗粒结构独特
- **Fuji**：冷色调，颗粒相对细腻
- 经验丰富的观众可通过颗粒特征识别胶片品牌

## 四、数字模拟方法

### 4.1 噪声模型类型

| 模型 | 特点 | 用途 |
|------|------|------|
| 高斯噪声 | 正态分布，最常用 | 基础颗粒模拟 |
| Perlin Noise | 连续渐变，自然感 | 纹理生成 |
| Simplex Noise | Perlin 改进版 | 高效纹理生成 |
| 椒盐噪声 | 黑白极值点 | 特殊效果 |

### 4.2 关键参数

```
Intensity (强度)  : 颗粒明显程度
Response (响应)   : 颗粒对亮度的反应曲线
Size (大小)       : 颗粒尺寸
Color (颜色)      : 颗粒是否带有色偏
```

### 4.3 实现要点

1. **时间变化**：每帧颗粒需有细微变化，避免静态感
2. **亮度响应**：暗部颗粒通常比亮部更明显
3. **色彩偏移**：彩色颗粒需模拟染料云的多色特性
4. **分辨率适配**：颗粒大小需随分辨率调整

## 五、视频编码中的 Film Grain

### AV1 编码标准处理流程

```
原始视频 (含 Film Grain)
        ↓
预处理降噪 (去除 Film Grain)
        ↓
编码器压缩 (降噪后视频)
        ↓
Film Grain 参数提取 → 写入码流
        ↓
解码端合成 (还原 Film Grain)
```

**优势**：保留胶片质感的同时提高压缩效率

## 六、艺术价值

### 6.1 视觉效果
- **复古风格**：模拟老电影质感
- **画面细节强化**：在特定条件下增强微弱信号检测 (随机共振)
- **电影感塑造**：增加影像的"胶片感"

### 6.2 心理感知
- 颗粒增加画面"有机感"和"温度"
- 减少数字图像的"过度清晰"和"塑料感"
- 在适度噪声条件下，图像视觉质量可达更优效果
